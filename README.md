# Underwriting Guide DB (Disease × Benefit Matrix)

본 저장소는 질병심사매뉴얼 화면에서 수집한 결과를 기반으로,
"질병명 → 심사기준 → 상품/급부별 인수 가능 여부"를 조회할 수 있는 DB를 제공합니다.

## 데이터 범위 / 보안
- 본 DB(`uw_knowledge.db`)는 다음 정보만 포함합니다.
  - 질병명, 심사기준, (상품/급부) 인수 가능 여부 및 관련 텍스트
- 개인정보, 계약정보, 고객식별정보는 포함하지 않습니다.

## ⚠️ Notice

본 저장소에 포함된 데이터는  
**내부 교육 및 테스트 목적의 언더라이팅 가이드용 데이터**입니다.

- 실제 인수 여부는 반드시 최신 약관, 공식 LAP, 개별 심사 결과를 기준으로 합니다.
- 본 데이터는 상품 급부별 인수 가능 여부 요약 정보이며,
  개인식별정보(PII)는 포함되어 있지 않습니다.
- 기준은 변경될 수 있으며, 본 저장소의 데이터는 참고용입니다.

## 구성
- `db/uw_knowledge.db` : 조회용 SQLite DB (GitHub 배포 대상)
- `app/` : 조회용 앱(예: Streamlit/FastAPI)
- `scripts/` : 수집/정제/DB빌드 스크립트
